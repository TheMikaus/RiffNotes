# RiffNotes v0.6.8

This release introduces improved song-section labeling workflows and marks section automation as experimental while the matching pipeline continues to be tuned.

## Highlights

- Added per-take section labeling options from the fingerprint menu:
  - Label sections (unique labels)
  - Label sections (allow repeats)
- Added bulk section-labeling options for matching takes and selected practices.
- Improved section alignment quality with partial-aware subsequence alignment and stronger duration/boundary constraints.
- Added a section labeling confidence slider in Preferences.
- Marked song section automation as experimental in project documentation.
