# RiffNotes v0.5.0

This release focuses on section editing UX, fingerprint-assisted title/section workflows, and safer sync operations.

## Highlights

- Improve section timeline editing with smoother drag/resize behavior and better edge handles.
- Add timeline interaction upgrades (gap/split interactions, drag preview feedback, section operations).
- Expand section workflows with color auto-assignment and millisecond-preserving section edits.
- Add take deletion workflow with confirmation and project refresh.
- Extend fingerprint workflow with low-confidence handling, accept/don't-know review flow, remembered suggestions, and skip re-evaluation for accepted matches.
- Align accepted master sections onto recordings using fingerprint-based placement instead of direct timestamp copying.
- Add upload file checklist (all selected by default) before sync.
- Add changed-only sync mode and optional delete-propagation mode for upload/download.
