# RiffNotes v0.3.0

This release captures the current Windows-focused practice review workflow after the waveform, notes, sections, playback processing, sync, and fingerprinting iterations.

## Highlights

- Waveform playback with cached waveform generation.
- Waveform zoom from 1x to 4x in 0.5x steps from the playback controls.
- Point notes, range notes, and per-track song sections.
- Clickable notes/sections that jump playback to the relevant time.
- Section lane display, section looping, and draggable section boundaries.
- Practice review view for seeing notes across the selected practice.
- Remembered band folder, practice, and per-practice selected recording.
- Volume boost up to +15 dB, left/right mute, mono fold-down, and Windows output-device selection.
- Cached processed playback and export as WAV or MP3.
- WAV/WAVE to MP3 conversion with recording metadata updates.
- Masters folder support and fuzzy fingerprint suggestion review.
- Manual Google Drive-style sync through a selected local sync folder.
- Practice-specific cache clearing.

## Verification

- `flutter test`
- `flutter build windows --no-pub`

## Notes

This is still experimental software. Windows is the primary development target. Android/Kindle Fire support remains planned around playback, note review/editing, sections, and sync.
